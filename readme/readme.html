Custom template for your stream!

OBS / StreamLaps and you are using SimToolKitPro!!
than you need this.

How wil it work ?

open 'connection.js' and you see lot of infomation;
the most importand are.
  • Protocol
    http protocal; you can change it to https if you using ssl;
  • schemaURI
    your local ip as default if you using other ip change it here!
  • schemaPORT
    You will find the port in Overlay(beta) as example http://localhost:51011/overlay2, the port is in this case : 51011
in the template section you can set true or false
to display your current information, in some template`s we have set
the values already correct for you; if you wabt to test it is on your self!
